# Java OOP Practice - Seoul Room Care

## Goal
This project is a Java OOP diagnostic practice project for reviewing object-oriented programming concepts.

## Main Conceppts
- Encapsulations
- Inheritance
- Polymorphism
- Abstraction
- Interface
- Enum
- Exception Handling
- Service and Repository separation

## Package Structure
- app: application entry point
- domain: core domain models
- services: business logic
- repository: data 
- exception: custom exceptions

## Proeject Domain
This project managers different types of rooms in Seoul, such as OneRoom, TwoRoom, and Officetel.
